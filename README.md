# nextboiler

This boilerplate has support for NextJS, Typescript, ESLint, prettier, styled-components and more can be added on. 

Begin by editing `src/pages/index.tsx`!

## To start the dev server:
```
yarn dev
```

## To build:
```
yarn build
```

## To start:
```
yarn start
```

## To lint:
```
yarn:lint
```

## To fix linting errors:
```
yarn fix
```
